ADR Template
============

Copy this file to ``NNNN-short-title.rst`` (next free number, lowercase, dashes)
and fill in the sections. Keep ADRs short — one screen if possible.

.. adr:: Short title in active voice
   :id: ADR-XXXX
   :status: draft
   :links: REQ-XXX, SPEC-XXX

   **Context.** What is the situation that forces a decision?

   **Decision.** What did we decide to do?

   **Consequences.** What becomes easier or harder because of this decision?
   Note any follow-up :need:`IMPL-XXX` work or new :need:`TEST-XXX` cases.

Coverage Report
===============

Generated by ``pytest --cov`` (see ``pyproject.toml``). The HTML report is
rendered into ``docs/reports/coverage_html/`` and is gitignored.

How to refresh
--------------

.. code-block:: bash

   pytest

The build will fail if line+branch coverage drops below **80%** (see
``--cov-fail-under=80``).

Latest summary
--------------

After running ``pytest`` locally, open
``docs/reports/coverage_html/index.html`` for the interactive view.
